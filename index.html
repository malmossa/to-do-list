A Simple to do list where you can add and remove to do items.

  • Bootstrap Layout.
  • Use of template literal to insert new list item with event delegation.
  • Use ForEach and if statment to filter the list items.
  • Confirm method before removing an item.
  • Toggle method to mark an item as completed.
